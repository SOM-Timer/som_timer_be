Som Timer Endpoints

GET '/api/timers/{timer_id}'

Response:

          

            {
              "id": 1,
              "work_interval": "25:00",
              "rest_interval": "5:00"
              "sound": "reverbSplash",
              "mood": true
            }

PUT '/api/timers/{timer_id}'

Request Body:

          

            {
              "work_interval": "30:00",
              "rest_interval": "7:00",
              "sound": "gong",
              "mood": false
            }

Response:

          

            {
              "id": 1,
              "work_interval": "30:00",
              "rest_interval": "7:00",
              "sound": "gong",
              "mood": false
            }

GET '/api/rand_exercise'

          
Required Query Params:

          

            * duration = 5:00, 7:00, 10:00
            * category = SOMATIC, MOVEMENT, MEDITATION
          

          
Ex:

          

            '/api/rand_exercise?duration=10:00&category=SOMATIC'
          

            
Response:

            

              {
                "id": 21,
                "url": "http://www.youtube.com/superrelaxinvibes",
                "duration": "5:00",
                "category": "SOMATIC"
              }
