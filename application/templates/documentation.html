Som Timer Endpoints

GET '/api/users/:user_id/timer'

Response:

          

            {
              "id": 1,
              "work_interval": "25:00",
              "rest_interval": "5:00"
              "sound": "reverbSplash",
              "mood": true,
              "user_id": 
            }

PUT '/api/users/:user_id/timer'

If a user already has a timer, the settings of that timer will be updated. If not, a new timer with the appropriate settings will be created.

Request Body:

          

            {
              "work_interval": "30:00",
              "rest_interval": "7:00",
              "sound": "gong",
              "mood": False
            }

Response:

          

            {
              "id": 1,
              "work_interval": "30:00",
              "rest_interval": "7:00",
              "sound": "gong",
              "mood": false,
              "user_id": 
            }

GET '/api/rand_exercise'

          
Required Query Params:

          

            * duration = 5:00, 7:00, 10:00
            * category = SOMATIC, MOVEMENT, MEDITATION
          

          
Ex:

          

            '/api/rand_exercise?duration=10:00&category=SOMATIC'
          

            
Response:

            

              {
                "id": 21,
                "url": "http://www.youtube.com/superrelaxinvibes",
                "duration": "5:00",
                "category": "SOMATIC"
              }
